# Deployment States

### Deployment States

Deployments go through a series of states. They are enumerated here and are useful if you are querying the status of a Deployment via the API.

| Name |
| :--- |
|  `Building`  |
| `Provisioning` |
| `Starting` |
| `Running` |
| `Stopping` |
| `Stopped` |
| `Error` |
